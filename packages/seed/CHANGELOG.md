# Changelog

All notable changes to `@cocapn/seed` will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2026-03-31

### Added

- **Core**: Soul-driven agent with memory, git awareness, and LLM integration
- **Core**: Multi-LLM support — DeepSeek, OpenAI, Ollama, and custom endpoints
- **Core**: Zero runtime dependencies — ships as pure TypeScript/JavaScript
- **Web chat**: Streaming chat interface with dark/light theme toggle
- **Web chat**: Markdown rendering and theme presets (ocean, forest, sunset, midnight, minimal)
- **Web chat**: Avatar support via soul.md frontmatter
- **Terminal chat**: Interactive CLI with slash commands
- **Soul**: Personality and behavior defined in `soul.md` (version-controlled)
- **Memory**: Fact store with key-value persistence
- **Git awareness**: Reads repo context for grounded responses
- **169 tests**: Full test suite covering core, web, soul, config, and themes
