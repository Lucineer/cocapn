# create-cocapn

[![npm version](https://img.shields.io/npm/v/create-cocapn.svg)](https://www.npmjs.com/package/create-cocapn)
[![MIT license](https://img.shields.io/badge/license-MIT-blue.svg)](LICENSE)

Zero-friction scaffolder for Cocapn agent instances.

## Use

```bash
npm create cocapn my-agent
cd my-agent
```

This sets up a new cocapn project with Git-backed memory, agent configuration, and a ready-to-run bridge.

## What You Get

- `cocapn.yml` — Bridge configuration
- `brain/` — Git-backed agent memory (facts, wiki, soul, procedures)
- `agents/` — Agent definitions
- `skills/` — Built-in skill stubs
