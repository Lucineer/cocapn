Submit a Plugin

Publish your plugin to npm and it appears here automatically.

  1. Create a cocapn-plugin.json manifest in your package root with your skills and permissions.
  2. Add "cocapn-plugin" to the keywords array in your package.json.
  3. Publish to npm with npm publish.
  4. It appears in this marketplace within a few minutes.
